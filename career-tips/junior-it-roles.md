# Junior IT Roles Overview

> Understanding entry-level positions in IT operations and support.

---

## 1. IT Support Specialist / Help Desk Technician

| Aspect | Detail |
|--------|--------|
| **Responsibilities** | Handle user tickets, troubleshoot hardware/software, reset passwords |
| **Key Skills** | Windows/Linux basics, Active Directory, ticketing systems, communication |
| **Tools** | ServiceNow, Jira, Freshdesk, Remote Desktop |
| **Growth Path** | Senior Support → System Administrator → IT Manager |

## 2. Junior System Administrator

| Aspect | Detail |
|--------|--------|
| **Responsibilities** | Manage servers, users, backups, updates, monitoring |
| **Key Skills** | Linux/Windows Server, scripting (Bash/PowerShell), networking, virtualization |
| **Tools** | Ansible, VMware, monitoring tools (Zabbix, Nagios) |
| **Growth Path** | Senior SysAdmin → DevOps Engineer → SRE |

## 3. NOC Technician (Network Operations Center)

| Aspect | Detail |
|--------|--------|
| **Responsibilities** | Monitor network and infrastructure, respond to alerts, escalate incidents |
| **Key Skills** | Networking, monitoring tools, incident response, documentation |
| **Tools** | SNMP monitoring, Grafana, PRTG, SolarWinds |
| **Growth Path** | NOC Engineer → Network Engineer → Cloud Engineer |

## 4. Cloud Support Associate

| Aspect | Detail |
|--------|--------|
| **Responsibilities** | Support cloud customers, troubleshoot cloud services, document solutions |
| **Key Skills** | AWS/Azure/GCP basics, networking, Linux, customer communication |
| **Tools** | AWS Console, Azure Portal, CloudWatch, CloudTrail |
| **Growth Path** | Cloud Engineer → Solutions Architect → Cloud Architect |

## 5. Junior DevOps Engineer

| Aspect | Detail |
|--------|--------|
| **Responsibilities** | Build CI/CD pipelines, manage infrastructure, automate deployments |
| **Key Skills** | Git, Docker, CI/CD tools, IaC (Terraform), scripting |
| **Tools** | Jenkins, GitHub Actions, Docker, Kubernetes, Terraform |
| **Growth Path** | DevOps Engineer → Senior DevOps → Platform Engineer |

---

## Skills That Transfer Across All Roles

- Linux command line
- Networking fundamentals
- Scripting (Bash, Python)
- Git and version control
- Documentation
- Troubleshooting methodology
- Customer communication
