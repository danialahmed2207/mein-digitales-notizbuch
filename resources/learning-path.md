# Learning Path & Resources

> Recommended resources and a structured path for IT operations and cloud engineering.

---

## Phase 1: Foundations (1–3 Months)

| Topic | Resources | Goal |
|-------|-----------|------|
| Linux Basics | Linux Journey (linuxjourney.com), Ubuntu Server docs | Navigate CLI confidently |
| Networking | Cisco Networking Basics, Professor Messer Network+ | Understand TCP/IP, DNS, DHCP |
| Bash Scripting | Bash Academy, Linuxize Bash Guide | Write basic automation scripts |
| Git | GitHub Skills, Pro Git book | Version control fluency |

## Phase 2: System Administration (3–6 Months)

| Topic | Resources | Goal |
|-------|-----------|------|
| Linux Server | LPIC-1 materials, Linux Foundation courses | Administer Linux servers |
| Windows Server | Microsoft Learn | Manage AD, Group Policy |
| Virtualization | VMware Hands-on Labs, VirtualBox docs | Build and manage VMs |
| Monitoring | Prometheus docs, Zabbix docs | Set up basic monitoring |

## Phase 3: Cloud & DevOps (6–12 Months)

| Topic | Resources | Goal |
|-------|-----------|------|
| Cloud Fundamentals | AWS Free Tier, Azure Free Account | Deploy and manage cloud resources |
| Docker | Docker docs, Docker Labs | Containerize applications |
| CI/CD | GitHub Actions docs, Jenkins docs | Build automation pipelines |
| IaC | Terraform tutorials, Pulumi docs | Manage infrastructure as code |

## Free Labs & Practice

| Platform | What to Do |
|----------|------------|
| **AWS Free Tier** | Launch EC2, S3, set up a VPC |

| **OverTheWire: Bandit** | Practice Linux via SSH challenges |
